    Subscription management

    Most of the content we consume is moving to digital format, and many companies consume content online. Small businesses are looking for tools to help employees share documents securely.

    The client

    Our client is an information technology support company well established. IT offers hosting services and computer infrastructure.

    Objectif

    Client wants a web application that manage users’ subscription and payment.

    About the Web application

    Web applications provide information about products, list subscription plans and prices, create user profiles, have user dashboards, and manage subscriptions and payments. It also provides a content management system – CMS and a management dashboard for managing users and their subscriptions.

    Requirements

    • Managing Users
    • Managing Subscription
    • Managing Payment
    • Managing Content (CMS)
    • Displaying Plans
    • Displaying Information (Product, pricing, About-us…)
    • Sign In/Up users
    • Dashboard users Profiles
    • Administrator dashboard
    • Notification and reminders (Quota Limit)
    • User friendly interfaces
    • Contact form
    • Search form

    Solution

    The web application is a Single Page Application SPA that uses a serverless CMS application to mange the content, a user management Api, a subscription management Api, an identity management system, a payment Api. It also connected through an Api to the core back-end wich is a the Files sharing management System.
